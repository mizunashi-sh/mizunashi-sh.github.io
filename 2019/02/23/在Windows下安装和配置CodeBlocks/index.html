在Windows下安装和配置CodeBlocks

Feb 23, 2019 1478

Power Station Of Art, Huangpu, Shanghai

注意:在不同操作系统之间移植C/C++程序可能需要修改既有代码
在Linux下进行C/C++编程是一个不错的选择，但是有时候使用Linux却并不方便–配置双系统比较麻烦，而使用虚拟机的话性能可能会不尽如人意。
好在Windows上也有很多不错的开发环境，CodeBlocks也提供Windows版本。因此，除了一些特殊情况，完全可以在Windows下使用CodeBlocks进行C/C++编程。
在Windows下安装和配置CodeBlocks与在Linux上稍有不同，下面是详细步骤。

安装MinGW

CodeBlocks默认不包括编译器，而需要使用事先已安装好的编译器。大多数Linux发行版都预装了C/C++编译器，所以CodeBlocks在Linux上安装后可以直接编写并编译程序。然而遗憾的是，Windows并没有预装C/C++编译器。
因此，需要自己安装合适的C/C++编译器，这里安装的编译器是MinGW。
MinGW(Minimalist GNU for Windows)，又称mingw32，是将GCC编译器和GNU Binutils移植到Win32平台下的产物，包括一系列头文件(Win32API)、库和可执行文件。GCC支持的语言大多在MinGW也受支持，其中涵盖C、C++、Objective-C、Fortran及Ada。
MinGW安装程序可以在这里下载。
打开安装程序后，点击Install安装，然后一路Continue即可。

安装成功后会自动弹出MinGW Installer Manager，桌面上也会出现相关图标。通过这个管理器可以安装或移除想要的软件包。
安装C/C++编译器，需要进行如下操作。
在左侧菜单打开All Packages->MinGW->MinGW Base System->MinGW Compiler Suite，在需要安装的软件包的条目中，单击右键，点选菜单中的Mark for Installation以勾选。
在这里需要勾选以

  • mingw32-binutils-
  • mingw32-gcc-
  • mingw32-g++-

为前缀的所有包，如图所示：

已经默认勾选的包请维持现状，不要取消勾选。
在编写程序时，调试器也是必要的，安装调试器，需要在左侧菜单打开All Packages->MinGW->MinGW Base System->MinGW Source-Level Debugger，并勾选这个栏目下的所有包，如下图所示：

在完成勾选之后，点击上方菜单栏的Installation->Apply Changes，在弹出的窗口中单击Apply。

然后就会开始下载。

等待进度条走完即安装成功，安装成功后应如下图所示：

也可以安装其它合适的C/C++编译器。

安装和配置CodeBlocks

CodeBlocks可以在这里下载，本次下载的是codeblocks-17.12-setup.exe。
打开安装程序，一路Next即可。

安装完成后，桌面和开始菜单中会出现CodeBlocks图标，点击运行。
第一次运行会检测到已经安装了编译器，点击Set as default，保存设置。

然后，就可以进入主界面，愉快地在Windows下使用CodeBlocks进行coding啦。

Windows下CodeBlocks的使用方法与Linux下的也是大致相同的，因此如果已经掌握如何使用Linux下的CodeBlocks的话，就完全不需要重新学习如何使用这个IDE了。
